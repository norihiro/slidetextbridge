Slide Text Bridge¶

Key Features¶

  • Capture live slideshow text from Microsoft PowerPoint and LibreOffice Impress

  • Powerful filters such as JMESPath filter selecting by meta-data, line break adjustment, text replacement by regular expression.

  • Send text to live streaming applications such as OBS Studio and vMix.

See Galley for visual examples.
