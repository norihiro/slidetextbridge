Filter Step: regexÂ¶

This step filters lines.

  • type: regex

Parameters:

patterns (list, required):

Set the patterns to search and replace.

patterns[*].p (str, required):

The pattern to search.

patterns[*].r (str, required):

The text to be replaced with.

YAML Example:

steps:
  - type: ppt
    placeholder_only: false

  - type: regex
    patterns:
          - p: '\<This\>'
            r: That
          - p: '\<Those\>'
            r: These

  - type: print
