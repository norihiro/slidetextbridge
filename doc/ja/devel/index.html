Slide Text Bridge¶

特徴¶

  • Microsoft PowerPointとLibreOffice Impressからライブスライドショーのテキストをキャプチャ

  • JMESPath を使用したメタデータによるテキスト選択、改行調整、正規表現によるテキスト置き換えなどの強力なフィルタ

  • OBS StudioやvMixなどのライブストリーミングアプリケーションにテキストを送信

画像による例については Galley を参照してください。
