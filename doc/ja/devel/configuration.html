設定ファイルリファレンス¶

設定ファイルは、テキスト処理パイプラインの steps を定義する YAML ドキュメントです。

トップレベルの構造¶

トップレベルには、各ステップの一覧を保持する steps キーを設定してください。

steps:
  - ... # Step 1
  - ... # Step 2
  - ... # and so on

ステップの種類¶

各ステップは type キーを持つオブジェクトです。追加のキーが必要になる場合があります。

利用可能な種類の完全なリストについては、 ステップの種類 を参照してください。
