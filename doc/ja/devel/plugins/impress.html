Input Step: impress¶

LibreOffice Impress のプレゼンテーションからテキストを読み取ります。 入力ソースとして設計されており、通常はパイプラインの最初のステップです。

  • type: impress

パラメータ:

host (string, 任意):

Impress のホスト名を設定します。デフォルトは localhost です。

port (integer, 任意):

LibreOffice がリスンしているポート番号を設定します。デフォルトは 2002 です。

pipe_name (string, 任意):

LibreOfficeが待機しているパイプ名を設定します。デフォルトは未設定です。

poll_wait_time (float, 任意):

ポーリングごとの待ち時間を秒単位で設定します。デフォルトでは0.1秒です。

ソケットでの接続

通常、設定ファイルにパラメータを指定する必要はありません。 ただし、ホストまたはポートを変更して、リモートホストに接続することができます。

steps:
  - type: impress
    host: localhost
    port: 2002
    poll_wait_time: 0.1

ソケットを使用するには、以下のようにlibreofficeを起動してください。

libreoffice '--accept=socket,host=localhost,port=2002;urp;'
パイプでの接続

pipe_name が指定されている場合、LibreOfficeへの接続は名前付きパイプを介して行われます。以下は設定例です。

steps:
  - type: impress
    pipe_name: slidetextbridge_pipe
    poll_wait_time: 0.1

ソケットを使用するには、以下のようにLibreOfficeを起動してください。

libreoffice '--accept=pipe,name=slidetextbridge_pipe;urp;'
