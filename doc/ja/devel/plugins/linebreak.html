Filter Step: linebreak¶

改行を調整します。

  • type: linebreak

パラメータ:

これらのパラメータは、図形や行を結合する方法を制御します。

shape_delimiter (string, optional):

図形の間の区切り文字を設定します。デフォルトは \n です。

line_delimiter (string, optional):

図形内の行の間の区切り文字を設定します。デフォルトは \n です。

strip (boolean, optional):

行を結合する前に、各行の先頭と末尾のスペースを取り除きます。デフォルトは true です。

例えば、複数の行を1つにしたい場合は、 line_delimiter を ' ' (単なるスペース) に設定してください。

以下のパラメータは、行の分割や結合を制御、すなわち改行位置の調整を行います。CJK(中国語、日本語、韓国語)に最適です。

split_long_line (integer, optional):

長い行を分割する際のしきい値。デフォルトは 0 で、分割を行いません。

split_nowrap (string, optional):

分割する際に、直前で分割を行わない文字を設定します。ご使用の言語での句読点を設定してください。デフォルトは .,"' ですが '、。」』,.' などを設定することを推奨します。

split_nowrap_allow_overflow (boolean, optional):

split_nowrap で設定した文字がオーバーフローする場合、オプションはオーバーフローを許可するかどうかを制御します。デフォルトは true です。

joined_column_max (integer, optional):

短い行を結合する際のしきい値。デフォルトは 0 で、結合を行いません。

join_by (string, optional):

短い行を結合する際の区切り文字。デフォルトは ' ' です。

処理の順序:

本文は以下の順序で処理されます。

  1. 長い行を分割

  2. 短い行を結合

  3. 図形内のに行を結合

  4. 図形を結合

YAML設定例:

steps:
  - type: ppt

  # Adjust line breaks
  - type: linebreak
    shape_delimiter: '\n'
    line_delimiter: ' '
    strip: true

  # This step prints the result
  - type: stdout
