Input Step: ppt¶

Microsoft PowerPoint のプレゼンテーションからテキストを読み取ります。 入力ソースとして設計されており、通常はパイプラインの最初のステップです。

  • type: ppt

パラメータ:

placeholder_only (boolean, optional):

プレースホルダの図形のみを選択します。デフォルトでは true です。

poll_wait_time (float, optional):

ポーリングごとの待ち時間を秒単位で設定します。デフォルトでは0.1秒です。

プレースホルダとは

プレースホルダとは、スライドマスターまたは新しいスライドページが作成されたときに存在する図形を意味します。 プレースホルダにすべてのテキストを挿入することをお勧めします。

しかし、他のすべてのテキストを取得したい場合は、 placeholder_only を false に設定します。 これにより、すべての図形が処理されます。

YAML設定例:

steps:
  # This step reads from PowerPoint. The text is then passed to the next step.
  - type: ppt
    placeholder_only: true
    poll_wait_time: 0.1
