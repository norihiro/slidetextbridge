Output Step: obsws¶

obs-websocket プラグインを使用して OBS Studio のテキストソースにテキストを送信します。

OBS Studio バージョン 28.0.0 以降で、obs-websocket がプリインストールされていることを確認してください。

  • type: obsws

パラメータ:

url (string, 必須):

OBSインスタンスのWebSocket URLを設定します。デフォルトは ws://localhost:4455/ です。

password (string, 任意):

OBS WebSocket サーバーの設定で設定したパスワードです。 パスワードを設定していない場合は、空白のままにするか、省略することができます。

source_name (string, 必須):

OBS Studio 内のテキストソースの正確な名前。例えば、 Text (GDI+) や 字幕 など。

YAML設定例:

steps:
  - type: ppt

  # This step sends the text to the 'Live Text' source in OBS.
  - type: obsws
    url: 'ws://localhost:4555/'
    password: 'your_secret_password'
    source_name: 'Text (GDI+)'
