Input Step: openlp¶

オープンソースの教会用プレゼンテーションアプリケーションである OpenLP からテキストを読み取ります。

  • type: openlp

パラメータ:

host (string, 任意):

Impress のホスト名を設定します。デフォルトは localhost です。

port (integer, 任意):

OpenLP がリッスンしているポート番号を設定します。デフォルトは 4316 です。

port_ws (integer, 任意):

WebsocketでOpenLPがリッスンするポート番号を設定します。デフォルトは port_ws + 1、すなわち 4317 です。

YAML設定例:

steps:
  # This step reads from OpenLP
  - type: openlp
    host: localhost
    port: 4316

使用方法¶

まず、OpenLP でリモートコントロール機能を有効にする必要があります。 Settings / Configure OpenLP に移動し、 Remote Interface タブを開きます。

Port number と Remote URL を確認してください。

説明¶

OpenLPにはいくつかのタイプがあります。Songs、聖書 、 カスタムスライド からテキストを取ります。 プレゼンテーション や 画像 などのその他の型は無視されます。
