Output Step: webserver¶

ブラウザー上にテキストを表示できるよう、ウェブサーバーをホストします。テキストはリアルタイムで更新されます。

  • type: webserver

パラメータ:

host (string, optional):

リスンするホスト。デフォルトは localhost です。 0.0.0.0 を設定すると、別のホストからも表示できます。

port (integer, optional):

リスンするポート番号。デフォルトは 8080 です。

index_html (string, optional):

HTMLファイルを上書きします。

script_js (string, optional):

JavaScript の内容を上書きします。

style_css (string, optional):

スタイルシートの内容を上書きします。

YAML設定例:

steps:
  - type: ppt

  # This step sends the text to the 'Live Text' source in OBS.
  - type: webserver
    host: 0.0.0.0
    port: 8080
    style_css: |
      body {
        font-family: sans-serif;
        font-size: 32pt;
      }

次に、ブラウザを開き、http://127.0.0.1/ (またはコンピュータのIPアドレス) に移動します。

注釈

  • vMix からページを開いた場合、背景は透明になっています。詳細は vMix User Guide を参照してください。
