Filter Step: stdout¶

コンソールにテキストを出力します。

  • type: stdout

パラメータ:

json (boolean, optional):

JSON 形式で表示します。デフォルトは false です。

page_delimiter (string, optional):

スライドページ間の区切り文字。デフォルトは \n\n (すなわち空の行) です。

YAML設定例:

steps:
  - type: ppt

  # This step prints all shapes with their properties.
  - type: stdout
    page_delimiter: '\n\n'
    json: true
