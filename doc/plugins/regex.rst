Filter Step: `regex`
====================

This step filters lines.

- **type: regex**

**Parameters:**

:``patterns`` (list, required): Set the patterns to search and replace.
:``patterns[*].p`` (str, required): The pattern to search.
:``patterns[*].r`` (str, required): The text to be replaced with.
